# ECHO-lab
# 反内耗：拒信粉碎机

一个基于 Python + Flask 的求职情绪支持与复盘助手。

## 功能特点

- 情绪安抚，支持多种语言风格（高情商 / 幽默嘴替 / 硬核教练）
- 拒信反向吐槽与心理减压
- 基于简历与岗位要求的客观复盘
- 4 周反弹学习计划
- 替代岗位推荐
- 拟真聊天与硬核复盘双模式

## 快速开始

1. 创建并激活虚拟环境。
2. 安装依赖：

```bash
pip install -r requirements.txt
```

3. 创建环境变量文件：

```bash
cp .env.example .env
```

4. 在 `.env` 中填写 AI 平台配置：

- `AI_API_KEY`: 你的密钥
- `AI_BASE_URL`: `https://aiping.cn/api/v1`
- `AI_MODEL`: 你的平台支持的模型名称

5. 启动服务：

```bash
python app.py
```

浏览器访问 http://127.0.0.1:5000

## 关于 59 元 AI 额度的配置说明

在 `AI_BASE_URL` 中设置平台地址，在 `AI_MODEL` 中设置模型名。例如：

- `AI_BASE_URL=https://aiping.cn/api/v1`
- `AI_MODEL=GLM-4.7-Flash`

## 用户操作指南（简短）

1. 启动后打开 `http://127.0.0.1:5000`，右上角默认在“控制台”。
2. 在“情绪粉碎”中输入岗位要求、简历、拒信内容，选择 AI 语言风格后点击“开始分析：原因 + 计划”。
3. 在“硬核复盘”中选择“八股”或“算法手撕”，再选编程语言，输入卡点问题后点击“AI 深度诊断与靶向学习”。
4. 在“拟真聊天”中可与 AI 连续对话，支持切换“理性教练 / 高能嘴替”，可一键清空当前会话。
5. 右上角切到“历史记录”可查看你的分析与对话摘要，支持清空历史。
6. 左侧“重置段位”可将连击、段位和情绪护盾恢复到初始状态。
